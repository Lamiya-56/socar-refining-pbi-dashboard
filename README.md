# SOCAR Refining Operations & Efficiency Dashboard

## 📊 Project Overview

This Power BI dashboard provides a comprehensive analysis of **SOCAR's refining operations**, focusing on operational efficiency, cost management, and environmental impact. It serves as a strategic tool for monitoring key performance indicators (KPIs) across different processing types, including Oil, Gas, and Chemical processing.

## 🚀 Key Features

* **Operational KPIs:** Tracking total costs ($57.35M$), average process duration ($38$ months), and total workforce ($18K$).
* **Efficiency Analysis:** Visualization of process efficiency categories (High, Medium, Low, Very High) and their impact on operational costs.
* **Environmental Impact (ESG):** Monitoring $CO_2$ intensity and emission levels against targets ($Goal: 0.35$).
* **Safety Monitoring:** Overview of total safety incidents categorized by process type.
* **Financial Insights:** Detailed breakdown of operating expenses (OPEX) by suppliers and process dates.

## 🛠️ Tools Used

* **Power BI:** For data visualization and dashboard creation.
* **DAX:** Used for calculated measures like $CO_2$ intensity and efficiency percentages.
* **Power Query:** For data transformation and cleaning.

## 📈 Dashboard Structure

1. **Main Dashboard:** High-level overview of production, safety, and workforce.
2. **Cost Analysis (Xərclər üzrə):** Deep dive into operational expenditures and supplier performance.
3. **Process Efficiency:** Analysis of how different processing units perform relative to their efficiency goals.
